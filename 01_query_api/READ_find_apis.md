## Finding APIs

üïí *Estimated Time: 10 minutes*

---

**REST APIs** power many of the apps on the internet, but most of them are safeguarded by **API keys** and subscriptions.
So where can we find useful and helpful APIs? Below, we index several useful, free APIs you may consider for your projects, broken down into groups.

Find more at this helpful Github repository: [public-apis/public-apis](https://github.com/public-apis/public-apis)

---

## Free, Public APIs

### üèõÔ∏è Government (General Topics)

- [**United States Census American Community Survey API**](https://www.census.gov/data/developers/data-sets/acs-1year.html): Free REST API describing population demographic characteristics for every census block, block group, tract, county, state, and more in the US, from 2005-present.

- [**World Bank Indicators API**](https://datahelpdesk.worldbank.org/knowledgebase/topics/125589-developer-information): provides programmatic access to time series development data and metadata for countries worldwide, collected and maintained by the World Bank.

- [**USASpending.gov API**](https://api.usaspending.gov/docs/intro-tutorial): The USAspending API (Application Programming Interface) allows the public to access comprehensive U.S. government spending data. The data include spending on awards (e.g., who received federal contracts or grants, geographic breakdowns, agency breakdowns, and much more). The data also include account-level such as federal employee compensation. You can learn more about the data and the federal law that requires it to be publicly accessible (the DATA Act) at http://fedspendingtransparency.github.io.

- [**UK Police Data API**](https://data.police.uk/docs/): Official Data API for UK Police, including content such as: neighbourhood team members, upcoming events, street-level crime and outcome data, and nearest police stations.

### üöå Transportation

- [**Bay Area Rapid Transit API**](http://api.bart.gov/): Official API for California's Bay Area Rapid Transit program, delivering data on stations and predicted arrivals for rapid transit. **Requires API Key.**

- [**Boston MBTA Transit API**](https://www.mbta.com/developers/v3-api): Official API for Boston, MA metro area's transit system. Reports stations and predicted arrivals for buses, metro, and trains. **Requires API Key.**

- [**NHTSA Vehicle API**](https://vpic.nhtsa.dot.gov/api/): The US National Highway Traffic Safety Administration's (NHTSA) Product Information Catalog Vehicle Listing (vPIC) has an API to share data on Vehicles and their specifications. The vPIC Dataset is populated using the information submitted by the Motor Vehicle manufacturers through the 565 submittals. All the information on how a VIN is assigned by the manufacturer is captured in this catalog and used to decode a VIN and extract vehicle information. **No API key needed.**

- [**GBFS Real-Time BikeShare Status APIs**](https://github.com/MobilityData/gbfs/blob/master/systems.csv): A dataset of >1000 different cities' General Bikeshare Feed Specification (GBFS) APIs, which can be queried to find currently available bikes, stations, etc., in any of these cities' systems.

### üå± Environment

- [**SafeCast API**](https://api.safecast.org/en-US/home): Citizen science crowd-sourced API for sharing radiation monitoring. Started in Tokyo after the Fukushima disaster in 2011; now reports data from sensors around the world.

- [**PurpleAir API**](https://community.purpleair.com/t/about-the-purpleair-api/7145): World's largest crowd-sourced air pollution sensor tracking database. Query results in realtime. **Requires API key.**

- [**USGS Earthquake Catalog API**](https://earthquake.usgs.gov/fdsnws/event/1/): Earthquakes data real-time, provided by the USGS Earthquake Hazards Program.

- [**USGS Water Services API**](https://waterservices.usgs.gov/): Water quality and level info for rivers and lakes throughout the US. Incudes streamflow, height, groundwater levels, historical data, and water quality, among other statistics.

- [**EPA Office of Water Services API**](https://watersgeo.epa.gov/openapi/waters/): The Watershed Assessment, Tracking & Environmental Results System (WATERS) unites water quality information previously available only from several independent and unconnected databases. WATERS provides a suite of interoperable services that expose components that perform complex analysis against supporting strategic datasets, such as NHDPlus, StreamCat and the Reach Address Database.

- [**EPA Air Quality System (AQS) API**](https://aqs.epa.gov/aqsweb/documents/data_api.html): This API is the primary place to obtain row-level data from the EPA's Air Quality System (AQS) database. If you would like aggregated or visual data from AQS, or large files of pre-extracted data, AirData is the place to start. *Primarily for historical air quality data. Always 6 months out of date.*

- [**BreezoMeter Pollen Forecast**](https://docs.breezometer.com/api-documentation/pollen-api/v2/): Daily forecast pollen conditions for a specific location. **Register for API Key.**

- [**BreezoMeter Wildfire Tracker API**](https://docs.breezometer.com/api-documentation/wildfire-tracker-api/v1/): Provides ongoing wildfire updates for your areas of interest, including area monitoring and precision tracking. **Register for API Key.**

- [**BreezoMeter Weather API**](https://docs.breezometer.com/api-documentation/weather-api/v1/): Provides weather conditions, such as temperature, wind speeds, humidity, precipitation, etc. for a specific location, up to five days prior **Register for API Key.**

- [**OpenAQ Air Quality API**](https://docs.openaq.org/using-the-api/quick-start): Open air quality data, with worldwide coverage. **Register for API Key.**

- [**Weatherstack**](https://weatherstack.com/?utm_source=Github&utm_medium=Referral&utm_campaign=Public-apis-repo-Best-sellers): Retrieve instant, accurate weather information for any location in the world in lightweight JSON format. **Register for API key.**

- [**UK Urban Observatory API**](https://urbanobservatory.ac.uk/): Real-time urban sensor readings on air quality, buildings, people, vehicles, water, weather, and more.

### üè• Health

- [**Centers for Medicare & Medicaid Services (CMS) Provider Data API**](https://data.cms.gov/provider-data/): Query provider data on hospitals, doctors and clinicians, physicial office visit costs, nursing homes, rehab services, and more. **Register for API key.**

- [**openFDA Device Adverse Events API**](https://open.fda.gov/apis/device/event/how-to-use-the-endpoint/): The U.S. Food and Drug Administration (FDA) regulates medical devices in the United States. Medical devices range from simple tongue depressors and bedpans to complex programmable pacemakers and laser surgical devices. In addition, medical devices include in vitro diagnostic products, such as general purpose lab equipment, reagents, and test kits, which may include monoclonal antibody technology. Certain electronic radiation emitting products with medical application and claims meet the definition of medical device. Examples include diagnostic ultrasound products, x-ray machines, and medical lasers.

- [**openFDA Food Adverse Events API**](https://open.fda.gov/apis/food/event/): The openFDA food, dietary supplements, and cosmetic adverse event API returns data from the Center for Food Safety and Applied Nutrition Adverse Event Reporting System (CAERS), a database that contains information on adverse event and product complaint reports submitted to FDA for foods, dietary supplements, and cosmetics. The database is designed to support CFSAN's safety surveillance program.

- [**openFDA Device Recall API**](https://open.fda.gov/apis/device/recall/): The openFDA's official API for reporting recalls of mdeical devices. A recall is an action taken to address a problem with a medical device that violates FDA law. Recalls occur when a medical device is defective, when it could be a risk to health, or when it is both defective and a risk to health.

- [**openFDA Drug Product Labeling API**](https://open.fda.gov/apis/drug/label/) Drug manufacturers and distributors submit documentation about their products to FDA in the Structured Product Labeling (SPL) format. The openFDA drug product labeling API returns data from this dataset.

- [**openFDA Drug Shortages API**](https://open.fda.gov/apis/drug/drugshortages/): Drug Shortages can occur for many reasons, including manufacturing and quality problems, delays, and discontinuations. Manufacturers provide FDA most drug shortage information, and the agency works closely with them to prevent or reduce the impact of shortages. When a shortage is listed as current on the Drug Shortage Database, the FDA is aware of the supply situation and is working with the manufactures on efforts to mitigate the supply disruption. FDA also continue to work with manufacturers on shortage prevention efforts for drugs not yet listed on the Drug Shortage Database.

- [**openFDA Drug Adverse Event Overview**](https://open.fda.gov/apis/drug/event/): The openFDA drug adverse event API returns data that has been collected from the FDA Adverse Event Reporting System (FAERS), a database that contains information on adverse event and medication error reports submitted to FDA.

### ‚ö° Energy 

- [**OpenEI API**](https://apps.openei.org/services/): Access complex utility rate structure information (across all sectors) for most U.S. utility companies from the National Utility Rate Database. This information is collected and quality controlled on a continual basis by Illinois State University on behalf of DOE and housed within the OpenEI.org platform.

- [**PVWatts API**](https://developer.nrel.gov/docs/solar/pvwatts/v6/): Energy production projections for photovoltaic energy systems based on system location and data. Produced by the National Renewable Energy Labs (NREL). **Register for API Key.**

- [**UK National Grid API**](https://data.nationalgrideso.com/): Open data from Great Britain's Electricity System Operator	

- [**Carbon Intensity API**](https://carbon-intensity.github.io/api-definitions/#carbon-intensity-api-v2-0-0): Official Carbon Intensity API for Great Britain developed by National Energy System Operator (NESO). The API provides an indicative trend of regional carbon intensity of the electricity system in Great Britain (GB) up to 2 days ahead of real-time.

### üì∞ Media & Press

- [**The Guardian API**](https://open-platform.theguardian.com/): free instant access to all of the Guardian's articles, images, audio, and videos dating back to 1999. **Register for an API key**.

- [**The New York Times API**](https://developer.nytimes.com/apis): search and query a dozen APIs about New York Times content, including article search, books, archive, movie reviews, publication times, and more. **Register for an API key**.

- [**The Associated Press API**](https://developer.ap.org/): programmatically access data from the Associated Press, including content, metadata, elections tallies, and more.
