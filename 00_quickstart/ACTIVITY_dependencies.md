## ‚úÖ Your Checklist

- [ ] Complete [`ACTIVITY_git.md`](ACTIVITY_git.md) to install **Git** and **Git Bash**
- [ ] Complete [`ACTIVITY_software.md`](ACTIVITY_software.md) to install **Python**, **R**, and your **IDE** (VS Code, Cursor, or RStudio)
- [ ] Update [`.bashrc`](.bashrc) with your Python and R paths
- [ ] Open **Git Bash** terminal and load your settings with `source .bashrc`
- [ ] Run `source setup/dependencies.sh` to load **R** and/or **Python** dependencies
- [ ] Start coding!
